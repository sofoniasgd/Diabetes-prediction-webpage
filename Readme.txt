Diabetes prediction webpage made using Flask and python

python libraries required:
Flask
pandas
pickle
scikit-learn


HOw to USE:

1: open a terminal on a python enabled computer
2: navigate to this project folder where flaskp.py is located(eg cd c:\users\...)
3: type and run "python flaskpage.py"
4: open web browser and type in the address: 'http://127.0.0.1:5000/'

If you are having module/library not found errors:
1. open a terminal inside the project folder
2. activate the virtual enviroment created for this project by running 'venv\Scripts\activate' command
3. install all the dependencies using pip
    pip install --upgrade pip
    pip install flask
    pip install scikit-learn
    pip install pandas
4. finaly run python flaskpage.py", open a browser and in the address bar
     type 'http://127.0.0.1:5000/'(this will also be displayed in your terminal window) and hit enter.
